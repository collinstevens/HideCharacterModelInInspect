# Hide Character Model in Inspect

When you inspect a player, the 3D character model is automatically hidden.

## Install

Install directly from [CurseForge](https://legacy.curseforge.com/wow/addons/hide-character-model-in-inspect), or download and drop the `HideCharacterModelInInspect` folder into:

```
World of Warcraft/_retail_/Interface/AddOns/
```

Then restart WoW or `/reload`.
